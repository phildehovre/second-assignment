How to Phlay:

Behind the scenes is a mystery word (randomly generated through an API call). Type your guess and press "enter". The letters that are in the right place will show in dark blue, the ones that are not in the right place, in light blue, the ones that do not exist in the mystery word in white.

About Phwirdle

Phwirlde is word guessing game and it is my implementation of a very popular game owned by a historically popular American newspaper.

This implementation is written entirely in vanilla Javascript. I challenged myself to code this application in two afternoons without the help of any tutorials. Sure, I could have used React and some fancy libraries. But where is the fun in that? The purpose of this exercise was to build a production-level browser game under very specific time and tooling constraints.
